## How to Edit This Project

There are several ways to work on your application:

### 1. Using Your Preferred IDE

You can work locally using VS Code or any other IDE.

**Requirements**: Node.js & npm installed. You can install Node.js via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd aloomia-ecommerce-hub

# Step 3: Install dependencies.
npm install

# Step 4: Start the development server with auto-reloading.
npm run dev
